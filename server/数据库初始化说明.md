# 校园安全管理系统 - 数据库初始化说明

## 快速开始

### 1. 初始化数据库

双击运行 `初始化数据库.bat`，或者在命令行中执行：

```bash
cd server
mysql -u root -p1314520xh < database-init.sql
```

### 5. 启动系统

数据库初始化完成后，可以启动系统：

**方式一：分别启动**
```bash
# 启动后端
cd server
node index.js

# 启动前端（新开命令行窗口）
cd ..
npm run dev
```

**方式二：一键启动**
双击运行 `..\一键启动.bat`

### 6. 访问系统

### 7. 常见问题

**Q: 数据库连接失败？**
A: 确保MySQL服务已启动，运行 `net start MySQL80`

**Q: 如何修改数据库密码？**
A: 修改 `server/config/db.js` 中的 `password` 配置

**Q: 如何重置数据库？**
A: 删除 `campus_guard` 数据库，重新运行初始化脚本

### 8. 技术栈

- **后端**: Node.js + Express + MySQL
- **前端**: Vue3 + Vite + Pinia + Element Plus
- **数据库**: MySQL 8.0

### 9. 注意事项

1. 确保MySQL服务正在运行
2. 确保数据库用户有足够的权限
3. 初始数据中的密码都是测试密码，生产环境请修改
4. 所有操作都会实时同步到数据库
